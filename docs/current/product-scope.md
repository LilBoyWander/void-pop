# Product Scope

Void Pop is a client-only, short-session browser arcade game. The player moves
horizontally, fires vertical shots, splits bouncing spheres, avoids hazards, and
clears five progressively harder sectors.

## Implemented

- Start, playing, paused, hit, level-clear, game-over, victory, and restart states
- Keyboard and touch controls
- Ball gravity, bouncing, splitting, and size-based behavior
- Platforms, hazards, and a later shooter-ball variant
- Score, survival contribution, combos, accuracy, lives, and local high score
- Responsive sci-fi UI, Canvas effects, fullscreen, and procedural audio

## Not approved by default

Accounts, network services, cloud saves, leaderboards, multiplayer, monetization,
analytics, new dependencies, heavy game engines, native packaging, and major new
game modes require explicit human approval.

The long-form `void-pop.md` is design reference material; unimplemented ideas in
it are not automatically in scope.
